## Codebase Patterns
- **FastAPI router pattern**: Routers are defined in `app/api/<name>.py` with `router = APIRouter(prefix="/<name>", tags=["<name>"])`, then imported and registered in `app/main.py` via `api_v1.include_router(<name>_router)`
- **Schema pattern**: Schemas use `SQLModel` base class from sqlmodel, with `Field()` for descriptions and examples
- **Test pattern**: Tests use `pytest.mark.asyncio` for async endpoint tests, direct function calls (not HTTP client)
- **Run commands**: Use `uv run` for all Python commands (mypy, pytest, etc.)
